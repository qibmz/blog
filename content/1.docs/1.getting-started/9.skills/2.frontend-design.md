# Frontend Design

Frontend Design 是一个专注于**前端界面设计与实现**的技能，帮助开发者在 Claude Code 中构建高质量、美观的 UI 组件和页面。

## 核心能力

### 设计系统驱动

- 使用设计系统和组件库（如 Nuxt UI、Tailwind CSS）构建一致的用户界面
- 遵循既定的设计语言和视觉规范
- 确保组件在不同场景下的一致性和可复用性

### 响应式与可访问性

- 移动端优先的响应式布局设计
- 键盘导航和屏幕阅读器支持
- 颜色对比度和焦点管理

### 动画与交互

- 使用 `motion-v`（Motion for Vue）声明式处理动画
- 滚动触发、悬停效果、页面过渡
- 微交互增强用户体验

### 组件开发

- 使用 Vue 3 Composition API 构建组件
- Props、Slots、Events 的合理设计
- 组件状态管理和数据流

## 技术栈

该技能适配以下技术栈：

| 类别 | 技术 |
|------|------|
| 框架 | Nuxt 4 / Vue 3 |
| UI 库 | Nuxt UI (U 前缀组件) |
| CSS | Tailwind CSS |
| 动画 | motion-v (Motion for Vue) |
| 图标 | Iconify (UIcon) |
| 图片 | Nuxt Image (NuxtImg) |

## 设计原则

1. **内容优先** — 设计服务于内容，而非相反
2. **渐进增强** — 基础功能不依赖 JS，增强体验可依赖 JS
3. **性能意识** — 懒加载、图片优化、代码分割
4. **一致性** — 遵循项目既有的设计模式和编码规范

## 使用方式

在 Claude Code 对话中输入 `/frontend-design` 启动该技能，然后描述你的 UI 需求。
