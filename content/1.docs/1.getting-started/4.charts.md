## echarts
- [网站链接](https://echarts.apache.org/){target="_blank"}
- 一个基于 JavaScript 的开源可视化图表库

## vue-echarts

- [网站链接](https://github.com/apache/echarts){target="_blank"}
- 专为 Vue.js 设计的 ECharts 组件，提供响应式和交互式图表

## ucharts

- [网站链接](https://www.ucharts.cn/v2/#/){target="_blank"}
-   PC、H5、APP、小程序、Vue、Taro
## uni echarts

- [网站链接](https://uni-echarts.xiaohe.ink/){target="_blank"}
-   适用于 uni-app 的 ECharts 组件 与 Vue ECharts 近乎一致的使用体验

## klinecharts

- [网站链接](https://v9.klinecharts.com/){target="_blank"}
- 一款轻量级的金融图表库，主要用于绘制K线图和其他金融图表
