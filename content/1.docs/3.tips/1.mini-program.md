# 小程序开发注意事项

在使用 uni-app 开发小程序时，会遇到一些平台特定的问题和限制。以下是常见的注意事项和解决方案：

## 支付宝小程序

### Swiper 组件高度问题

支付宝小程序中 swiper 组件默认设置的高度可能无效，需要手动设置 `adjust-height="none"` 来解决高度异常问题。

```html
<!-- 示例 -->
<swiper adjust-height="none">
  <swiper-item>内容1</swiper-item>
  <swiper-item>内容2</swiper-item>
</swiper>
```

## 微信小程序

### 作用域插槽不支持

微信小程序不支持作用域插槽（Scoped Slots），在开发过程中应避免使用该特性，或提供替代方案。

### XML 响应体解析问题

微信小程序的接口请求不支持直接解析 XML 响应体，如果需要处理 XML 数据，需要在业务逻辑中手动解析。

```javascript
// 错误示例 - 不能直接处理 XML
uni.request({
  url: 'https://example.com/api.xml',
  success: (res) => {
    // res.data 不会自动解析 XML 格式
  }
})

// 正确做法 - 手动解析 XML 或要求后端返回 JSON
```

### 普通链接打开二维码配置

在微信小程序中配置普通链接打开二维码时，在体验版调试时必须在测试链接中添加相应的链接才能正常跳转。

### 接口无法访问问题

微信小程序体验版中可能会出现接口无法访问的情况，但在打开调试模式后就恢复正常。这种问题通常是以下原因之一：

1. 请求的接口域名为 HTTP 协议而非 HTTPS
2. 接口域名未在微信公众平台后台配置为 request 合法域名

**解决方案：**
- 确保接口使用 HTTPS 协议
- 在微信公众平台「开发管理」->「开发设置」中配置合法的 request 域名
